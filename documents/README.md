# Your CV document

Place your CV file here so the **Download my CV** button on your portfolio works.

**File to add:**
- **cv.pdf** â€” Your resume/CV in PDF format.

**Steps:**
1. Save or export your CV as a PDF.
2. Name the file **cv.pdf** (lowercase).
3. Put it in this `documents` folder.

The Resume page will link to `documents/cv.pdf`. If you prefer a different filename (e.g. `resume.pdf`), put that file here and update the link in `index.html`: find `href="documents/cv.pdf"` and change it to `href="documents/yourfilename.pdf"`.
